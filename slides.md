# 蓝印花布——南通的非遗瑰宝
### 江海文化与民间艺术的交织
刘赫 · 媒体2201

---

# 🧭 引言

- 蓝印花布与濠河、狼山并称“南通三宝”，记录江海儿女的生活记忆
- 2006 年列入国家级非物质文化遗产，保留古老的夹缬防染智慧
- 以国家级代表性传承人吴元新为代表的工匠群体守护完整技艺链
- 它不仅是一块布，更是一段历史、一种精神
- 融合江南民间智慧与审美意象，在衣被、礼俗与文创中延续生命力

# 起源与历史

---

# 📜 起源与历史 · 概览

- 蓝印花布技艺源于古代夹缬、防染传统，秦汉时期已有植物靛染记载
- 唐宋时期江南逐渐形成专门印染作坊，花版图谱初具体系
- 明清民间商业繁荣，南通二甲镇成为“蓝印花布之乡”
- 清末实业家张謇将民间作坊与近代工厂结合，促成产业化
- 改革开放后重启传统作坊，形成生产、展示、体验三位一体的传承模式

---

# 📜 起源与历史 · 发展脉络

| 时期 | 发展特点 | 关键节点 |
|------|-----------|---------|
| 秦汉 | 植物靛染萌芽 | 文献记载“蓝草染布” |
| 唐宋 | 防染技术成熟、作坊集聚 | 江南贡品中出现印花布 |
| 元明清 | 江南市镇繁荣，花版体系完备 | 《天工开物》详述蓝靛制作 |
| 近现代 | 工业化冲击与保护并存 | 张謇办厂、乡镇合作社复兴 |
| 当代 | 非遗+文创驱动新生 | 博物馆、研学、品牌联名 |

---

# 📍 主要产区分布

- 江苏南通：余西古镇、二甲镇沿袭手工流程，是国家级传承人集中地
- 浙江桐乡：乌镇、濮院等地依托桑蚕丝织业发展蓝印花布体验工坊
- 湖南邵阳：侗、瑶族地区保留蓝印花布婚嫁习俗，形成南派风格
- 江南一带：苏州、扬州、湖州等地习得技艺，构建跨区域生产网络
- 海外传播：旅欧华人将蓝印花布带入艺术展场，拓宽影响力

---

# 🏞️ 江南文化背景

- 江南水乡气候湿润，适合棉花与蓼蓝生长
- 河港纵横，方便布匹、花版与灰料的运输与交流
- 印染业与家庭纺织天然结合，一家有机具即可开坊
- 民间图案承载吉祥与信仰，寄托对丰年、婚嫁、寿礼的祝福
- 寺庙会市与海港贸易促进蓝印花布向周边地区扩散

# 🪡 工艺流程 · 概览

- 选棉纱、漂练、整经，为防染打底
- 取蓝染料（蓼蓝草 → 蓝靛），调制石灰灰汁
- 雕版印浆（豆粉+石灰+石膏）并晾干定型
- 多次染色 → 氧化 → 晾晒，循环加深色泽
- 洗浆显花，捣浆去污并整烫
- 冰裂纹顺着温度与风向自然生成，留下手工痕迹

# 🌿 第一步：取蓝染料

- 蓝印花布染料来自蓼（liǎo）蓝草，秋季收割后堆沤发酵
- 通过“打蓝靛”工序过滤、风干，得到可长期保存的蓝靛块
- 调入石灰、草木灰制成靛缸，维持碱性与还原环境
- 天然植物染料环保、色泽持久，可反复复原为深靛蓝

---

# 🪵 第二步：雕版印浆

- 雕刻花版是整个工艺的灵魂，常用梨木或樟木以防变形
- 工匠先按传统纹样描图，再以阳刻、阴刻镂空线条
- 花版覆盖在白布上，用刮板均匀刮涂由豆粉、石灰、石膏调成的防染浆剂
- 印浆后自然晾干，形成挺阔纹理，为后续染色打下基础

---

# 🫧 第三步：染色

- 将印好浆的布浸入靛缸，使布面充分吸附染液
- 反复染 6～8 次甚至更多，以达到层层晕润的色阶
- 每次染色约 20 分钟，出缸后抖散、氧化 30 分钟
- 染匠需要随时打理染缸活性，维持靛液的还原性

---

# 🍃 第四步：氧化

- 自然透气氧化让靛蓝遇氧呈现由黄绿转蓝的变化
- 曝晒过程中需避开暴晒与阴雨，以保持色泽均匀
- 工匠通过手感与色差判断最佳氧化时间
- 合理的翻动、抖甩能避免色层沉积造成色花

---

# 💧 第五步：洗浆显花

- 将布料放入清水或溪水中反复冲洗，去除防染浆
- 传统上使用木槌捶打，使浆料彻底脱落、布面柔软
- 蓝底白花、白底蓝花开始分明，纹样细节完全显现
- 手工印浆的“冰裂纹”也在此阶段逐渐浮现

# 🧊 冰裂纹的独特美感

- 防染浆干燥时随温度、湿度自然开裂
- 形成不规则细裂纹，宛如霜花或冰纹
- 这是机器印花无法复制的自然肌理
- 每幅布都有独一无二的“指纹”，收藏家据此辨识手工精品

# 🌿 蓝底白花

- 以一块花版完成印浆，线条简洁有力
- 图案多为梅兰竹菊、蝙蝠、莲花、寿桃等吉祥元素
- 常用于帐幔、被里、桌裙，凸显纯净朴素的生活美学
- 冰裂纹在深蓝背景上更显立体，适合现代设计改造

---

image: https://blue-calico-slides-r2.xlyytcy04.site/White%20Background%20with%20Blue%20Patterns.jpeg

# 🌿 白底蓝花

- 需两块花版套印，第二块“盖版”遮盖连接处
- 工艺更复杂，对版面对位及浆料厚度要求更高
- 适合表现写意山水、书法题字等大面积图案
- 呈现多层次视觉效果，常见于礼仪服饰与陈设品

---

# 🧭 图案寓意

- 🌸 花草 → 吉祥如意，象征“岁岁平安”“四季长春”
- 🐦 动物 → 平安长寿，蝙蝠寓意“福”，喜鹊寓意“报喜”
- ☯️ 文字纹样 → 祝福祈愿，常见“寿”“喜”“福禄”组合
- 🏠 日常器物 → 反映农耕与渔牧劳作场景，记录民俗记忆
- 融合民间信仰与生活理想，是口耳相传的视觉语言

---

# 🖌️ 经典纹样故事

- “莲生贵子”以莲花、莲蓬、童子组合，寄托多子多福的愿望
- “五蝠捧寿”将蝙蝠与寿字相拥，寓意五福临门
- “八吉祥”受佛教文化影响，表达对平安吉祥的祈愿
- “耕读传家”场景描绘男耕女织，传递勤俭持家的价值观

# 👤 张謇与产业化

- 张謇（1853—1926），近代实业家、教育家
- 创办大生纱厂，推动棉纱与印染的一体化发展
- 1905 年设立“蓝印花布作坊”，扶持乡间女工就业
- 建立花版档案、培训制度，让传统技艺进入现代市场
- 其倡导的实业兴国理念，为非遗保护留下制度基因

# 👥 当代传承与社区

- 以吴元新等国家级代表性传承人为核心，持续整理传统花版
- 南通二甲镇成立传承人工作室，形成“师徒制”口传心授机制
- 校企合作开设非遗工坊，让大学生参与设计与染制实践
- 线上直播、开放日等活动吸引更多年轻人关注与参与

# 🏛️ 蓝印花布博物馆

- 建于 1996 年，位于濠河畔老厂区
- 馆藏实物 3000 余件，涵盖花版、染缸、服饰与文献
- 设有开放式体验区，让游客亲手印浆、染色
- 展示传统工艺与现代设计结合，是国家级非遗传承与推广基地
- 通过研学课程、主题展览，让非遗走进课堂与社区

# 🌍 国际传播与影响

- 多次参加国内外非遗与设计展览，向世界展示江南印染美学
- 深受设计师与艺术家喜爱，与家居、服饰品牌推出跨界合作
- “蝶恋花”“耕读图”等系列成为讲述东方故事的视觉符号
- 传统工艺与当代设计结合，成为跨文化交流的媒介
- 海外华人社群通过讲座、工作坊推广蓝印花布美学

---

# 🧠 小结

- 蓝印花布不仅是一块布，更是一段历史与文化的载体
- 它承载江南水乡的生活智慧，也映射中国手工艺的审美哲学
- 在传承人与设计师的共同努力下，传统工艺焕发当代活力
- 南通借此向世界讲述独特的江海故事，持续输出文化自信
